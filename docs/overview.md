# Обзор проекта Smart City Management Dashboard

## Назначение
Smart City Management Dashboard — это демонстрационный MVP (прототип), разработанный для визуализации состояния города, выявления критических ситуаций и предоставления рекомендаций на основе искусственного интеллекта (OpenAI).

## Основные возможности
- **Интерактивный Dashboard**: Отображение ключевых KPI города (Транспорт, Экология, Общий риск).
- **Мониторинг Транспорта**: Отслеживание загруженности дорог, средней скорости и инцидентов по районам.
- **Мониторинг Экологии**: Отслеживание индекса качества воздуха (AQI) и уровня мелкодисперсных частиц (PM2.5, PM10).
- **Система Уведомлений**: Автоматическое выявление проблемных ситуаций на основе заданных порогов.
- **AI-Аналитика**: Генерация кратких сводок и управленческих рекомендаций с помощью OpenAI GPT.
- **Демо-сценарии**: Возможность переключения между тремя состояниями города (Норма, Умеренная проблема, Критическая ситуация) для демонстрации логики системы.

## Локализация
Проект полностью локализован на русский язык, включая интерфейс, мок-данные и ответы ИИ.
